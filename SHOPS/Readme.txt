
LIFF APP ที่มีให้บริการ
* หมายถึงต้องมี
1. menu   : ใช้ในการแสดงผลเมนูแต่ละ 1 เมนู เมืี่อ Click *
2. menulist   : ใช้ในการแสดงผลให้ลูกค้าเลือกตามประเภท Menu อาหาร
3. menusearch : ใช้สำหรับ ให้ลูกค้า Search Menu กรณี เมนูเยอะหลายรายการ
4. cart       : ใช้แสดงรายการสินค้า หรือ menu ที่เลือกในร้าน *
5. addcart    : ใช้เพื่อทำรายการเพิ่ม เมนูนั้นๆ เข้าสู่ระบบ shopping cart *
6. promotion  : ใช้เพื่อแสดงสินค้าที่หน้าหลัก สำหรับการตลาด
7. orderlist  : ใช้เพื่อแสดงรายการสั่งซื้อ ทั้งหมดในมุมมองของลูกค้า

อัจเพิ่ม
8. menu_detail : แสดงรายการสินค้า 1 รายการ เพื่อดูรายละเอียด  และสั่งซื้อได้
